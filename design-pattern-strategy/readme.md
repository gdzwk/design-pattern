# 策略模式


## 模式定义
定义一个算法族，其中的每个算法均封装为具体的策略，根据业务场景的不同，适用不同的策略。

## 模式结构
* **context** : 环境类 (策略决策)
* **strategy** : 策略抽象类
* **specificStrategy** : 具体策略 

## 策略特点
* 优点
  * 符合开闭原则
  * 方便管理算法族
* 缺点
  * 需要生成大量的类
  * 客户端需要理解具体的策略作用，才能做出合适的选择
  
## spring中策略模式的应用
* Resource (资源访问策略)
* InstantiationStrategy (实例化策略)
